# NeoVim Setup

A Neovim config based on [LazyVim](https://lazyvim.org).

## Getting Started

> If there's a current neovim setup, it's wise to back it up.

```bash
mv ~/.config/nvim ~/.config/nvim.bak
```

```bash
git clone git@github.com:lexedwards/nvim-setup ~/.config/nvim
```

## Language Support

* Go
* Javascript
* Rust
* Typescript
